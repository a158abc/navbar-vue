# navbar-vue
> Nuxt.js project
> 使用nuxt搭建的简单界面，包含一些vuex实例，axios实例，还使用了bootstrapUI。

#### 项目运行
>* npm i
>* npm run dev

#### 使用感受
> 感觉nuxt使用起来很方便，不用定义路由，它会根据文件夹的结构自动生成路由，很方便，运行也很流畅，最喜欢的是可以使用中间件来显示加载进度
